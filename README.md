# Astrosmash Clone
A clone of the Intellivision game Astrosmash, written in Ruby 2.1.2 using Gosu as the game engine.

Controls
--------
- Arrow Left / Right = Move Left / Right
- Space = Start Game / Fire
- R = Reset Game
- Esc = Quit
